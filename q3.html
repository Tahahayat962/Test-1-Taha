Old Fashioned Typesetting compared to Desktop Publishing, html, css. What is the difference?

Desktop Publishing

Desktop publishing is much faster than old fashioned typewriting as it is easier due to keyboards and screens with visual aids and brightness.

html

Html is even more different as it allows you to alter more aspects of the said page and it is online concerning webpages not real paper.

css

Css is the furthest from it as it is a coding language and someone used to old fashioned typewriting would not understand it, it is also for back end editing not real paper.

In conclusion, all these newer methods are vastly different from old fashioned typesetting, one could say they iteratively built upon it but it is still a very marked and major difference.
